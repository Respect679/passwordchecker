# 🔐 PasswordCheck

**PasswordCheck** is a Python-based desktop application that checks the strength of any password and tells you if it's been exposed in a real-world data breach. Built as part of a cybersecurity project, it features a modern black-themed UI, live strength checking, and integration with the Have I Been Pwned API.



## 🧠 Features

- ✅ Check password strength in real time
- ⚠️ Warns if the password is too common or weak
- 🌐 Checks if password has been leaked using" Have I Been Pwned"
- 👁️ Show/Hide password toggle
- 🖼️ Resizable cyber-themed background
- 📦 Can be exported as `.exe` for Windows users
